  • 获取单个订单的交易明细

    1.请求URL：http://www.blackhorse.pro/o/api/order/trades/{symbol}/{id}/{api_key}/{timestamp}/{sign_type}/{sign}

    2.请求类型：GET

    3.请求参数：

    参数 类型 是否参与签名 描述 示例
    symbol String 否 交易对名称 ETHQC
    id Integer 否 订单id 142455
    api_key String 是 用户apikey 5b092b11-001c-4e50-9d8e-00f9db752241
    timestamp long 是 时间戳精确到秒 1540619752
    sign_type String 是 签名类型 MD5
    sign String 否 具体查看签名认证 3cbbe236e86105b1a041cfe1fe01

    4.返回参数

    参数 类型 描述 示例
    state Integer 请求状态1成功，-1失败 1
    msg String 请求状态说明 success
    trades array 交易明细 略
    total int 返回的数量 1

    5.请求成功示例 

    {
 state: 1,
 msg: "success",
 trades: [{略}],
 total: 0
}

    6.请求失败示例 

    {
 state: -1,
 msg: "ILLEGAL_TIMESTAMP",
 trades: null,
 total: 0
}


  • 获取订单列表

    1.请求URL：http://www.blackhorse.pro/o/api/orders/{symbol}/{page}/{page_size}/{status}/{api_key}/{timestamp}/{sign_type}/{sign}

    2.请求类型：GET

    3.请求参数：

    参数 类型 是否参与签名 描述 示例
    symbol String 否 交易对名称 ETHQC
    page Integer 否 页码，起始页码为1 1
    page_size Integer 否 每页数量，最大20 20
    status String 否 0表示未完成（挂单中），1表示已完成（含已取消），2表示所有 1
    api_key String 是 用户apikey 5b092b11-001c-4e50-9d8e-00f9db752241
    timestamp long 是 时间戳精确到秒 1540619752
    sign_type String 是 签名类型 MD5
    sign String 否 具体查看签名认证 3cbbe236e86105b1a041cfe1fe01

    4.返回参数

    参数 类型 描述 示例
    state Integer 请求状态1成功，-1失败 1
    msg String 查询的交易对名称 ETHQC
    orders array 订单列表 略
    total Integer 返回的数量 1

    5.请求成功示例 

    {
 state: 1,
 msg: "ETHQC",
 orders: [{略},{略},{略} ],
 total: 0
}

    6.请求失败示例 

    {
 state: -1,
 msg: "ILLEGAL_TIMESTAMP",
 orders: null,
 total: 0
}

  • 创建交易订单

    1.请求URL：http://www.blackhorse.pro/o/api/order

    2.请求类型：POST

    3.请求参数：

    参数 类型 是否参与签名 描述 示例
    api_key String 是 用户apikey 5b092b11-001c-4e50-9d8e-00f9db752241
    o_no String 是 交易订单号，32位随机字符串 1540635044555-0.7886791948144788
    o_price_type String 是 价格类型：limit、market，分别代表限价单、市价单 limit
    o_type String 是 订单类型：sell，buy，分别代表卖、买 sell
    price BigDecimal 是 价格 136.8
    volume BigDecimal 是 数量 0.1
    symbol String 是 交易对名称 ETHQC
    timestamp long 是 时间戳精确到秒 1540619752
    sign_type String 是 签名类型 MD5
    sign String 否 具体查看签名认证 ade9e763f24276512c033f7cef76

    4.返回参数

    参数 类型 描述 示例
    state Integer 请求状态1成功，-1失败 1
    msg String 请求状态说明 success
    data long 订单id 2344221

    5.请求成功 

    {
 state: 1,
 msg: "success",
 data: 2344221
}

    6.请求失败 

    {
  "state":-1,
  "msg":"NO_SUFFICIENT_FUNDS"
}
